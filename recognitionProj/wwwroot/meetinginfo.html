How to Use the Meeting Info Page

Welcome to the Meeting Info page for administrators. Use this guide to efficiently create, manage, and review meetings along with their associated applications.

1. Creating and Selecting Meetings
  • Launch Meeting: Click the Launch Meeting Now button to create a new meeting session. This refreshes the meeting list and updates the dropdown with the latest meeting number.
  • Select a Meeting: Use the dropdown labeled Select a Meeting to choose a specific meeting. If none is selected, the page shows Pending Applications.
2. Viewing and Managing Pending Applications
  • Pending Applications Section: Displays institutions that have submitted applications but have not yet been assigned to a meeting.
  • Application Details: Each row shows:
    • Institution Name – Clickable to view detailed records.
    • Specialization, Submission Date, and Committee Recommendation.
    • Assigned Committee Member (supervisor).
  • Update Committee Member: For applications younger than 48 hours, select a new Committee Member from the dropdown and click Reassign to reassign.
3. Reviewing Meeting Applications
  • Meeting Applications Section: When a meeting is selected, a table displays all applications related to that meeting.
  • Detailed Data: View meeting number, institution (with clickable link for full records), specialization, submission date, assigned committee member, and committee recommendation.
  • Final Decision Submission: Two buttons – Submit Final Decision (English) and Submit Final Decision (Arabic) – let you process each application.
4. Using the Decision Modal
  • Modal Options: When you click a final decision button, a modal appears offering:
    • Recognize – finalizes the application as recognized.
    • Reject – declines the application.
    • Return Report to Committee Member – sends it back for further review.
    • Close – dismisses the modal without action.
  • Finalization Process: After selecting an option, the system saves your decision and redirects you to the corresponding final decision document based on your language choice.
5. Navigation
  • Return to Dashboard: Click the Return to Dashboard button at any time to go back to the main admin dashboard.

NOTE: for printing, make sure you press ctrl + p, click advanced options, more pringing options, set the page to a4, set the margins to none, remove the headers/footers, and enabling the background
